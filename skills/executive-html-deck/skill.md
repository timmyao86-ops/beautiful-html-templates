# Executive HTML Deck Skill

## Purpose
Create executive-grade editable HTML decks from PPTs, briefs, strategy memos, or research documents.

This skill specializes in:
- Product strategy reviews
- Competitive analysis
- Executive management reports
- Tender response materials
- Research synthesis
- Micro-mobility product analysis

## Default Configuration

Template: `blue-professional`

Output mode:
- Slide + scroll hybrid
- Editable HTML
- Shareable online link

Language:
- Default bilingual (CN/EN)

Audience:
- Executive / management level

## Inputs
Accepted formats:
- PPT / PPTX
- Markdown brief
- Strategy memo
- Product requirement document
- Screenshots
- Images
- Tables

## Workflow
1. Read input material
2. Extract information architecture
3. Do NOT convert slide-by-slide
4. Restructure into executive report
5. Select template (default: blue-professional)
6. Create preview version
7. Generate editable HTML
8. Deploy to GitHub outputs folder
9. Generate shareable link
10. Save reusable assets

## Output Requirements
Required features:
- Editable text blocks
- Adjustable font size
- Browser save
- Download modified HTML
- GitHub deploy-ready
- Mobile compatible
- Presentation + report readability

## Deployment Path
outputs/<project-name>/index.html

## Naming Convention
Project format:
<topic>-<year>-<type>

Example:
bolt7-competitive-review-2026
q3-product-strategy-2026
