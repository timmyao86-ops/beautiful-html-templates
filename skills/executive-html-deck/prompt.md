# Executive HTML Deck Prompt

## Invocation

Use:

`调用 Executive HTML Deck Skill`

or

`Use Executive HTML Deck Skill`

## Standard Input Format

```text
Use Executive HTML Deck Skill

Input:
[upload ppt or brief]

Goal:
[why this deck exists]

Audience:
[executive / investor / customer / product team]

Language:
CN / EN / bilingual

Template:
blue-professional

Output:
editable html

Requirements:
- preserve evidence images
- management storytelling
- github deploy
- downloadable html
```

## Default Behavior

Always:
- use blue-professional template
- preserve executive readability
- create editable html
- allow font resizing
- support browser save
- support html export
- create github share link
- bilingual support
- management-level narrative
